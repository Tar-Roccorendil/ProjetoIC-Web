Sistema de Numeração Primordial

Em geral, os sistemas de numeração posicionais representam um número como uma soma de potências de uma base constante β. Alguns sistemas, no entanto, têm base mista, e representam um número como uma soma de bases na sequência (β0, β1,β2,...).

O sistema primordial não é um desses sistemas. O princípio desse sistema é o Teorema Fundamental da Aritmética, que aprendemos ainda no meio Ensino Fundamental: todos os números naturais maiores que 1 podem ser expressos como um produto único de primos. O sistema primordial é apenas isso: uma representação da fatoração de cada número em primos. Para usá-lo, basta escrever na n-ésima posição o expoente do n-ésimo primo:

1DEC = 20(×30×50×...);
2DEC = 21(×30×50×...);
3DEC = 20×31(×50×70×...);
12DEC = 20SEX = 22×31(×50×70×...);
suas representações em primordial são 1DEC = 000000...PRI, 2DEC = 100000...PRI, 3DEC = 0100000...PRI e 20SEX = 12DEC = 2100000...PRI. Como todos os números naturais têm um número finito de fatores primos, e existem infinitos primos, todas as representações de um número natural
